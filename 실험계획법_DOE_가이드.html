실험계획법 (Design of Experiments, DOE)

개념 · 통계 이론 · 설계 방법 · Python/R 활용 · 제조·공정 적용 — 따라하기 쉬운 종합 가이드

0. 한눈에 보기

실험계획법(DOE)은 "어떤 입력 변수를, 어떤 조합으로, 몇 번 실험해야 가장 적은 비용으로 가장 신뢰할 수 있는 정보를 얻는가"를 통계적으로 설계하는 방법론입니다. 한 번에 한 인자만 바꾸는 OFAT 방식과 달리, 여러 인자를 동시에·균형 있게 바꿔 주효과와 교호작용을 모두 추정하고 최소 실험으로 최적 조건을 찾습니다.

핵심 메시지DOE = ① 효율적 실험 배치 + ② 통계 모형(ANOVA·회귀)으로 효과 분리 + ③ 반응표면 최적화. 이 세 가지가 하나의 흐름으로 연결됩니다.

이 문서는 정의 → 용어 → 통계 이론 → 설계 방법 → 절차 → 툴 사용 → 적용 사례 → 체크리스트 순으로 진행됩니다. 코드 블록은 우측 복사 버튼으로 바로 가져다 쓸 수 있습니다.

1. 실험계획법이란 무엇인가

1.1 정의

관심 반응(response, 출력)에 영향을 주는 인자(factor, 입력)들을 체계적으로 변화시키며 그 영향을 통계적으로 추정하기 위해 실험을 설계·수행·분석하는 방법론입니다. R. A. Fisher가 1920~30년대 농업 실험에서 체계화했고, 이후 산업 품질공학·공정 최적화로 확장되었습니다.

DOE의 목적은 셋으로 정리됩니다.

  • 스크리닝(Screening) — 다수 후보 인자 중 유의한 핵심 인자를 적은 실험으로 골라냄
  • 특성화(Characterization) — 핵심 인자의 주효과·교호작용을 정량 모형화
  • 최적화(Optimization) — 반응을 최대/최소화 또는 목표값에 맞추고 견고성 확보

1.2 왜 필요한가 — OFAT의 한계

OFAT(One-Factor-At-a-Time)는 직관적이지만 두 가지 치명적 약점이 있습니다. ① 같은 정밀도에 더 많은 실험이 필요하고, ② 한 인자의 효과가 다른 인자 수준에 따라 달라지는 교호작용을 구조적으로 탐지하지 못합니다.

구분OFATDOE(요인설계)
교호작용 추정불가능가능
동일 정밀도 대비 실험 수많음적음 (효율적)
실험 공간 탐색축을 따라 일부만공간 전체를 균형 있게
통계적 유의성 판단어려움ANOVA로 정량화
최적화 연결제한적반응표면법으로 직접 연결

1.3 좋은 실험의 3대 원칙 (Fisher)

  • 반복(Replication) — 동일 조건을 여러 번 실험해 실험 오차를 추정하고 정밀도를 높임
  • 랜덤화(Randomization) — 실험 순서를 무작위로 배치해 시간 드리프트 등 잡음의 편향을 방지
  • 블록화(Blocking) — 원료 로트·작업조처럼 알려진 변동 요인을 블록으로 묶어 오차에서 분리
기억하기 (R-R-B)Replication(정밀도) · Randomization(편향 제거) · Blocking(잡음 분리) — 세 원칙이 DOE 신뢰성을 떠받칩니다.

2. 핵심 개념과 용어

용어의미

수준 (Level)인자가 취하는 값. 2수준(±1), 3수준 등. 부호화로 −1/+1 표준화

xcoded = (xreal − center) / (range / 2)

3. 통계 이론 기초 (심화)

3.1 선형 모형과 최소제곱 추정

DOE 분석의 토대는 일반선형모형입니다. 관측 반응 y는 설계행렬 X, 모수 β, 오차 ε로 표현됩니다.

y = Xβ + ε,   ε ~ N(0, σ²I)

최소제곱(OLS) 추정량은 잔차제곱합을 최소화하여 닫힌 형태로 구해집니다. 직교설계에서는 XᵀX가 대각행렬이 되어 각 효과가 독립적으로 안정적으로 추정됩니다.

β̂ = (XᵀX)⁻¹ Xᵀy

2수준 요인설계에서 인자의 '효과'는 +1 평균에서 −1 평균을 뺀 값이며, 회귀계수는 그 절반입니다.

Effect = ȳ(+1) − ȳ(−1) = 2 × β̂

3.2 분산분석 (ANOVA)

전체 변동(SST)을 모형이 설명하는 변동과 오차 변동(SSE)으로 분해합니다.

SST = SS(model) + SSE,   SST = Σ(yᵢ − ȳ)²

평균제곱(MS)을 만들고 효과 MS를 오차 MS로 나눈 F비로 유의성을 판정합니다. p-값이 유의수준(보통 0.05)보다 작으면 유의합니다.

F = MS(effect) / MS(error),   p = P(F > Fobs)

주의F검정은 오차의 정규성·등분산·독립성 가정에 기반합니다. 분석 후 반드시 잔차도(정규확률도, 잔차 vs 적합값)를 확인하세요.

3.3 적합도와 모형 진단

  • R²·수정 R² — 모형이 설명하는 변동 비율. 수정 R²는 항 개수 패널티가 있어 과적합 비교에 유용
  • 예측 R²(Q²) — 새 데이터 예측력. 수정 R²와 크게 벌어지면 과적합 신호
  • 적합결여 검정(Lack-of-Fit) — 순수오차와 모형 잔차를 비교. 유의하면 항 부족(곡률 누락 등)
  • 잔차 진단 — 정규확률도(정규성), 잔차 vs 적합값(등분산), 잔차 vs 순서(독립성)

3.4 반응표면법(RSM)과 2차 모형

최적점 근방의 곡률을 표현하려면 2차 모형을 적합합니다(선형·교호작용·제곱항 포함).

ŷ = β₀ + Σβᵢxᵢ + Σβᵢⱼxᵢxⱼ + Σβᵢᵢxᵢ²

정상점(stationary point)은 기울기를 0으로 놓아 구하고, 헤시안 고유값 부호로 최대·최소·안장점을 판정합니다(정준분석). 다반응은 만족도(desirability) 함수로 동시 최적화합니다.

∂ŷ/∂x = 0  ⟹   xs = −½ B⁻¹ b

4. 주요 실험설계 방법

설계 선택은 목적(스크리닝/최적화)과 예산(실험 가능 횟수)의 함수입니다. 아래 흐름으로 고르세요.

  1. 인자가 많고 핵심만 거르고 싶다 → Plackett-Burman, 해상도 III 부분요인
  2. 주효과·교호작용을 깨끗이 보고 싶다 → 완전요인 또는 해상도 IV/V 부분요인
  3. 최적 조건(곡률 포함)을 찾고 싶다 → 반응표면(CCD, Box-Behnken)
  4. 잡음에 둔감한 견고 조건을 원한다 → 다구치(강건설계)
  5. 성분 비율 합이 1인 배합 문제다 → 혼합물 설계(Mixture)

4.1 완전요인설계 (Full Factorial)

k개 인자를 2수준 모든 조합(2ᵏ)으로 실험합니다. 모든 효과를 별칭 없이 추정해 정보량이 가장 풍부하지만, 인자가 늘면 실험 수가 급증(2⁵=32, 2⁷=128)합니다.

4.2 부분요인설계 (Fractional Factorial)와 해상도

완전요인의 일부(1/2, 1/4 …)만 실험합니다. '고차 교호작용은 거의 무시 가능'하다는 희소성 원리에 근거하며, 대신 일부 효과가 별칭됩니다.

해상도별칭 구조용도
III주효과 ↔ 2차 교호작용 별칭스크리닝. 교호작용 가정 시 주의
IV주효과 깨끗, 2차 교호작용끼리 별칭스크리닝~특성화 절충
V주효과·2차 교호작용 깨끗(3차와 별칭)특성화·정밀 분석
정의 관계예: 2(4-1)에서 D=ABC로 생성 → 정의관계 I=ABCD. 각 별칭은 I=ABCD에 곱해 구합니다(예: A=BCD).

4.3 Plackett-Burman 설계

실험 수가 4의 배수(8, 12, 20…)이며 많은 인자(최대 N−1개)를 적은 실험으로 스크리닝합니다. 해상도 III이라 주효과만 관심 있는 초기 단계에 적합합니다.

4.4 중심점과 곡률 검정

2수준 설계는 본질적으로 평면(선형)만 적합합니다. 중심점 실험을 추가하면 순수오차를 추정하고, 코너 평균과 중심 평균의 차이로 곡률을 검정할 수 있습니다. 곡률이 유의하면 RSM으로 확장합니다.

4.5 반응표면 설계: CCD vs Box-Behnken

항목중심합성설계 (CCD)Box-Behnken (BBD)
구성요인점 + 축점(±α) + 중심점변의 중점 + 중심점
수준 수5수준 (−α,−1,0,+1,+α)3수준 (−1,0,+1)
장점확장 용이, 회전성 설정 가능극단 조합 회피, 실험 수 적음
적합공정 최적화 일반극단 조건이 위험/고가일 때

4.6 다구치(Taguchi) 강건설계

제어 인자와 잡음 인자를 내·외측 배열로 함께 설계해, 잡음에 둔감하면서 목표 성능을 내는 조건을 찾습니다. 평균뿐 아니라 변동까지 줄이며 S/N비를 특성치로 씁니다.

S/N (망목) = 10·log₁₀(μ² / σ²)

4.7 혼합물 설계 (Mixture)

성분 비율 합이 1(100%)인 배합 문제(처방·조성)에 사용합니다. 인자가 독립적이지 않으므로 단체격자·단체중심 설계와 Scheffé 다항모형을 씁니다.

5. 실험 진행 절차 (단계별)

DOE는 일회성이 아니라 순차적 학습 과정입니다. '스크리닝 → 특성화 → 최적화 → 검증'으로 이어지며, 한 번에 전체 예산의 25% 이상을 쓰지 말라는 경험칙이 있습니다.

  1. 문제 정의 — 목표 반응·제약·현재 성능 명확화, 측정시스템(MSA) 점검
  2. 인자·범위 선정 — 공정지식·브레인스토밍으로 인자/수준/반응 결정
  3. 설계 선택 — 목적·예산에 맞는 설계 선정, 검정력(power) 확인
  4. 실험 수행 — 랜덤 순서로 실행, 조건·이상치 기록
  5. 분석 — ANOVA·회귀, 효과·교호작용, 잔차 진단
  6. 최적화 — RSM·만족도 함수로 최적 조건 도출
  7. 확인 실험 — 예측 조건 재실험으로 검증(confirmation run)
검정력(Power)설계 단계에서 '탐지하고 싶은 최소 효과 크기'를 정하고, 그 효과를 충분한 확률(보통 ≥0.8)로 검출할 반복 수를 미리 계산하세요. 실험 후 "데이터가 부족했다"를 막아줍니다.

6.2 설계 생성 (pyDOE3)

python
import numpy as np, pandas as pd
from pyDOE3 import ff2n, fracfact, ccdesign, bbdesign, pbdesign

# 2^3 완전요인설계 (부호화 -1/+1)
full = ff2n(3)                      # shape (8, 3)

# 2^(4-1) 부분요인설계: D = A*B*C  (정의관계 I=ABCD, 해상도 IV)
frac = fracfact('a b c abc')        # shape (8, 4)

# 중심합성설계(CCD): 2인자, 면중심(face-centered)
ccd = ccdesign(2, center=(1, 1), alpha='orthogonal', face='ccf')

# Box-Behnken: 3인자
bbd = bbdesign(3)                   # shape (15, 3)

# Plackett-Burman: 5인자 스크리닝
pb  = pbdesign(5)

부호화 행렬을 실제 단위로 환산하려면 중심값과 반범위(range/2)를 씁니다.

python
# 예: 온도 80~120℃, 농도 1~3M
center = np.array([100, 2.0]); half = np.array([20, 1.0])
real = full[:, :2] * half + center   # -1 -> 80℃, +1 -> 120℃

6.3 모형 적합과 ANOVA (statsmodels)

2인자 요인설계에 교호작용을 포함한 회귀모형을 적합하고 분산분석표·효과를 출력하는 전형적 흐름입니다.

python
import statsmodels.formula.api as smf
import statsmodels.api as sm

df = pd.DataFrame(ff2n(2), columns=['Temp', 'Conc'])
df['Yield'] = [68.1, 73.2, 71.0, 80.5, 79.8, 84.9, 83.1, 92.6]   # 측정 수율

# Temp*Conc 는 Temp + Conc + Temp:Conc 로 전개됨
model = smf.ols('Yield ~ Temp * Conc', data=df).fit()

print(model.summary())                  # 계수, t, p, R^2
print(sm.stats.anova_lm(model, typ=2))  # ANOVA 표 (SS, F, p)

effects = 2 * model.params              # 부호화 계수 -> 효과
print(effects)
해석 요령계수의 p<0.05면 유의. 부호화 모형에서 계수 절대값이 클수록 영향이 큽니다. Temp:Conc 계수가 유의하면 두 인자 사이에 교호작용이 존재합니다.

6.4 반응표면 적합과 최적화 (statsmodels + scipy)

python
from scipy.optimize import minimize

# 2차 모형 적합 (제곱항은 I(...)로 표현)
rsm = smf.ols(
    'Yield ~ Temp + Conc + Temp:Conc + I(Temp**2) + I(Conc**2)',
    data=df_ccd).fit()

# -Yield 를 최소화 = Yield 최대화
def neg_yield(x):
    g = pd.DataFrame({'Temp':[x[0]], 'Conc':[x[1]]})
    return -rsm.predict(g).iloc[0]

res = minimize(neg_yield, x0=[0, 0], bounds=[(-1, 1), (-1, 1)])
print('최적 부호값:', res.x, ' 예측 수율:', -res.fun)

7. 제조 / 공정 적용 사례

7.1 사례 개요 — 반응 수율 최적화

반응 공정에서 수율(%)을 최대화하려 합니다. 후보 인자는 온도·농도·교반속도·촉매량·시간 등 다수. 예산 제약상 순차적 DOE로 접근합니다.

단계설계목적·결과
① 스크리닝2(5-1) 부분요인(16run, 해상도 V) + 중심점 3온도·농도·시간 3개 유의, 곡률 유의 → RSM 필요
② 특성화/최적화3인자 중심합성설계(CCD, 회전성)2차 모형 적합, R²·예측R² 확인, 교호작용 규명
③ 최적화만족도(desirability) 최적화최적 조건 도출: 예측 수율 ≈ 93%
④ 확인 실험최적 조건 5회 반복실측 92.4±0.6% — 예측 구간 내, 검증 완료
python
# 1) 스크리닝: 부분요인 + 중심점으로 핵심 인자/곡률 확인
scr = smf.ols('Yield ~ (Temp+Conc+Stir+Cat+Time)', data=screen_df).fit()
# 유의 인자: Temp, Conc, Time  /  중심점 곡률 검정 -> 유의

# 2) RSM: 3인자 CCD에 2차 모형 적합
rsm = smf.ols('Yield ~ Temp + Conc + Time '
              '+ Temp:Conc + Temp:Time + Conc:Time '
              '+ I(Temp**2) + I(Conc**2) + I(Time**2)', data=ccd_df).fit()

# 3) 최적화: -Yield 최소화로 최대 수율 조건 탐색 (6.4절 방식)
# 4) 확인 실험으로 예측 검증
실무 포인트최적점이 실험 범위 경계에 걸리면 '진짜 최적'이 아닐 수 있습니다. 이때는 가장 가파른 상승 방향(steepest ascent)으로 영역을 이동한 뒤 다시 RSM을 수행합니다.

7.2 다른 제조 분야 확장

  • 사출/성형 — 금형온도·사출압·냉각시간으로 휨·치수 불량 최소화(망목, 다구치)
  • 용접/접합 — 전류·속도·압력으로 접합강도 최대화 및 변동 최소화
  • 도금/코팅 — 전류밀도·온도·시간으로 두께 균일도 확보(RSM 등고선)
  • 배합/조성 — 성분 비율 합=1 제약 → 혼합물 설계로 물성 최적 배합
