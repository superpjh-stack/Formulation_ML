## 6. 시스템 개요

### 6.1 시스템명
고려솔더 배합비율 최적화 AI 스마트공장 시스템 (Formulation ML)

### 6.2 핵심 기능
- **배합비율 최적화 AI**: ML 모델(GradientBoosting, RandomForest, XGBoost)을 이용한 최적 배합비율 자동 추천
- **품질 예측**: Sn/Ag/Cu/Pb 성분 조합에 따른 품질 점수 사전 예측
- **성분 편차 분석**: 목표값(Sn=62%, Ag=3%, Cu=0.5%) 대비 실측 편차 실시간 모니터링
- **AI 대시보드**: 생산/품질/설비/출하 현황 통합 뷰
- **AI Agent**: 자연어 기반 데이터 조회 및 의사결정 지원

### 6.3 기술 스택
| 구분 | 기술 |
|------|------|
| 프론트엔드 | Next.js 14, TypeScript, React 18 |
| ML 백엔드 | Python 3.10, FastAPI, scikit-learn, XGBoost, scipy |
| 배포 | Docker, Nginx, Ubuntu 22.04 |
| 데이터베이스 | PostgreSQL 15 |
