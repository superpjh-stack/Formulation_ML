## 1. 문서 개요

### 1.1 목적

본 문서는 Formulation ML 시스템(성분분석 데이터 기반 배합비율 최적화 ML 시스템)의 2026년 6월 운영 현황을 기록하고, 시스템 가용성, ML 모델 성능, 서비스 이용 현황, 품질 개선 효과 및 주요 이슈를 종합하여 운영 책임자 및 관련 부서에 보고하는 것을 목적으로 한다.

### 1.2 보고 기간

- 보고 기준일: 2026-06-19
- 보고 대상 기간: 2026년 6월 1일 ~ 2026년 6월 19일 (시범 운영 기간)
- 작성 주기: 월 1회 (매월 말일 기준 익월 5일 이내 발행)

### 5.2 주요 개선 사례

**사례 1 — SUP_B 공급사 전환 시 품질 안정화**

- 발생 시점: 2026년 6월 4일
- 상황: 기존 SUP_A에서 SUP_B로 원료 전환 시 품질점수 하락 (종전 평균 83점 → 77점)
- ML 시스템 대응: 공급사 전환 즉시 SUPPLIER_EFFECTS 보정 계수 자동 적용, 배합비율 재추천 (SN +0.3%, AG -0.1% 조정)
- 결과: 조정 후 품질점수 82점으로 회복, 불량 발생 없음
- 도입 전 대응: 숙련 작업자 경험 의존 → 평균 2~3 LOT 불량 발생 후 수동 보정

**사례 2 — DOE 시뮬레이션을 통한 온도 최적 구간 도출**

- 수행 시점: 2026년 6월 10일
- 목적: 용융 온도 240~270°C 구간에서 품질점수 최대화 조건 탐색
- 방법: CCD 방법 적용 (온도·시간 2인자, 9개 실험점)
- ML 시뮬레이션 결과: 온도 255°C, 시간 42분 조건에서 예측 품질점수 최대 (87.3점)
- 실적용 결과: 해당 조건 적용 LOT 5건 평균 품질점수 85.1점 (기존 평균 81.3점 대비 +3.8점)
- 기대 효과: 반복 실험 없이 9회 시뮬레이션으로 최적 조건 도출 (기존 방법 대비 실험 기간 약 4일 단축)

**사례 3 — 불량 예측으로 사전 조치**

- 발생 시점: 2026년 6월 16일
- 상황: 작업자가 계획 배합비율 입력 전 품질 예측 실행 → 예측 품질점수 71점 (불량 판정)
- ML 시스템 대응: 불량 위험 경고 표시 및 배합비율 재추천 제시
- 결과: 배합비율 조정 후 실행, 실측 품질점수 79점으로 불량 예방
- 도입 전 대응: 배합 실행 후 성분 분석까지 불량 여부 불인지 → 사후 처리
