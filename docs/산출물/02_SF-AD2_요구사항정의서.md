## 3. 용어 정의

| 용어 | 설명 |
|------|------|
| LOT | 동일 배합 조건으로 생산된 제품 단위 (예: LOT-2026-001) |
| 수율 | 전체 생산량 대비 합격 제품 비율 (%) |
| 품질 점수 | ML 모델이 예측하는 0~100 사이의 품질 지수 |
| 성분 편차 | 실측 성분값과 목표값(Sn=62%, Ag=3%, Cu=0.5%)의 차이 |
| XRF 분석 | 형광 X선 분석법 — 비파괴 성분 정량 분석 |
| SAC305 | Sn-Ag3%-Cu0.5% 무연 솔더 규격 |
| GradientBoosting | 앙상블 ML 알고리즘. 현 프로젝트 최고 성능 모델 |
| SLSQP | Sequential Least Squares Programming — 비선형 제약 최적화 알고리즘 |
| SN_TARGET | 배합 목표 Sn 비율 = 62.0% |
| AG_TARGET | 배합 목표 Ag 비율 = 3.0% |
| CU_TARGET | 배합 목표 Cu 비율 = 0.5% |
| RMSE | Root Mean Square Error — ML 회귀 모델 성능 지표 |
| R² | 결정계수 — 모델의 분산 설명력 (1에 가까울수록 좋음) |
