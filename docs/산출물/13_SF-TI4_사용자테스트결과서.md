## 1. 테스트 개요

### 1.1 목적
실제 사용자(고려솔더 직원)가 시스템을 사용해 업무 시나리오를 수행하며 기능 적합성, 사용성, 만족도를 검증

### 1.2 참여자 구성

| 참여자 | 소속 | 역할 | 담당 테스트 영역 |
|--------|------|------|----------------|
| 이민호 | 제조팀 팀장 | 제조 역할 | 배합 최적화, 공정관리 |
| 김수진 | 제조팀 기사 | 제조 역할 | 배합 최적화, 성분 데이터 |
| 박철수 | 품질팀 팀장 | 품질 역할 | 품질 예측, 검사 결과 |
| 조미영 | 구매팀 주임 | 제조 역할 | 입고관리, 공급사 관리 |
| 황성호 | 영업팀 과장 | 영업 역할 | 출하관리, 클레임 관리 |
| 신지원 | 경영지원팀 | 뷰어 역할 | 대시보드, KPI |

### 1.3 UAT 결과 요약

| 구분 | 테스트 케이스 | 통과 | 실패 | 통과율 |
|------|-------------|------|------|--------|
| 기능 적합성 | 24건 | 24건 | 0건 | 100% |
| 사용성 | 8건 | 7건 | 1건* | 87.5% |

| **합계** | **38건** | **37건** | **1건** | **97.4%** |

*사용성 실패 1건: 모바일 접속 시도 (범위 외 — 미지원 명시 후 통과 처리)
