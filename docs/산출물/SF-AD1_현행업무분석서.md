## 1. 문서 개요

### 1.1 목적

본 문서는 아모레퍼시픽 납땜 합금(솔더) 배합 공정의 현행 업무 실태를 체계적으로 분석하여, 스마트공장 보급·확산사업(선도형 제조혁신 지원사업)의 일환으로 추진하는 **Formulation ML 시스템** 구축의 타당성 및 개선 방향을 제시하는 데 목적이 있다.

현행 업무분석서는 다음 사항을 명확히 한다.

- 현재 납땜 합금 배합 공정의 업무 흐름 및 담당 조직·인원 파악
- 현재 품질 관리 방식의 한계 및 문제점 도출
- 데이터 수집·관리 실태와 활용 가능성 검토
- AI/ML 기반 시스템 도입을 통한 개선 방향 및 정량적 기대효과 제시

### 1.2 적용 범위

본 문서의 적용 범위는 다음과 같다.

- **대상 공정**: 납땜 합금(솔더) 배합 공정 — SN(주석)/AG(은)/CU(구리)/PB(납) 성분 배합비율 결정 및 생산
- **대상 조직**: 아모레퍼시픽 제조사업부 > 생산팀 > 납땜공정팀
- **분석 기간**: 현행 업무 기준 (2025년 1월 ~ 2026년 5월 운영 실적 기반)
- **제외 범위**: 솔더 배합 이후의 납땜 조립 공정, 완제품 검사 공정

### 1.3 용어 정의

| 용어 | 정의 |
|------|------|
| 솔더(Solder) | SN/AG/CU/PB 등을 일정 비율로 혼합한 납땜 합금 재료 |
| SN63/Pb37 | 주석 63%, 납 37% 비율의 유연(有鉛) 솔더 합금 |
| SAC 계열 | SN/AG/CU 3원 무연(無鉛) 솔더 합금 (예: SAC305: SN96.5/AG3.0/CU0.5) |
| LOT | 배합 단위. 1 LOT = 1회 배합 생산 배치 |
| 품질 점수 | 솔더 배합 품질을 0~100점으로 수치화한 내부 KPI. 75점 미만 불량 판정 |
| 성분 편차 | 목표 배합비율 대비 실제 투입 성분의 편차 (SN_TARGET: 62.0%, AG_TARGET: 3.0%, CU_TARGET: 0.5%) |
| 공급사 | 원재료(SN/AG/CU/PB) 납품업체. SUP_A(50%), SUP_B(30%), SUP_C(20%) |
| DOE | Design of Experiments (실험계획법). 다수의 변수를 체계적으로 설계하여 최적 조건을 탐색하는 통계적 방법 |
| SLSQP | Sequential Least Squares Programming. 제약 조건이 있는 비선형 최적화 알고리즘 |
| EDA | Exploratory Data Analysis (탐색적 데이터 분석). 데이터의 분포, 상관관계, 이상치 등을 시각적으로 탐색 |
| ML | Machine Learning (기계학습). 데이터로부터 패턴을 학습하여 예측 모델을 구축하는 기술 |

## 2. 기업 현황

### 2.1 기업 개요

| 항목 | 내용 |
|------|------|
| 회사명 | 아모레퍼시픽 주식회사 |
| 업종 | 화장품 및 생활용품 제조업 |
| 사업 분류 | 스킨케어·메이크업·헤어케어·바디케어 제품 제조 및 판매 |
| 주요 생산 품목 | 화장품 용기, 포장재, 전자 조립 부품 (납땜 조립 포함) |
| 납땜 공정 적용 제품 | 화장품 전자 기기(스킨케어 디바이스), 포장 설비 부품, 제조 자동화 장비 부품 |
| 스마트공장 추진 배경 | 제조혁신 가속화 및 품질 고도화를 위한 AI/데이터 기반 공정 최적화 추진 |

아모레퍼시픽은 화장품 및 생활용품 분야 국내 1위 기업으로, 국내외 다수의 제조 거점을 운영하고 있다. 최근 화장품 전자 기기(뷰티 디바이스) 시장의 성장과 함께 전자 부품 납땜 조립 공정의 중요성이 높아졌으며, 솔더 합금 배합 품질이 완성품 신뢰성에 직접적인 영향을 미친다는 내부 분석 결과에 따라 배합 공정 최적화가 핵심 개선 과제로 선정되었다.

### 2.2 조직 구조

납땜 합금 배합 공정은 아래 조직 체계 하에 운영된다.

```
아모레퍼시픽 제조사업부
└── 생산팀
    └── 납땜공정팀
        ├── 배합 파트 (배합 담당 기술원 3명)
        ├── 품질 파트 (품질 검사원 2명)
        └── 설비 파트 (설비 담당 기사 1명)
```

**조직별 주요 역할**

| 조직 | 담당 인원 | 주요 역할 |
|------|-----------|-----------|
| 납땜공정팀 팀장 | 1명 | 공정 전체 관리, 품질 기준 결정, 외부 공급사 관리 |
| 배합 파트 수석 기술원 | 1명 | 배합비율 결정 (경험 기반), LOT 배합 지시 |
| 배합 파트 기술원 | 2명 | 실제 배합 작업 수행, 원재료 투입 및 계량 |
| 품질 검사원 | 2명 | 배합 후 성분 분석 및 품질 검사, 불량 판정 |
| 설비 담당 기사 | 1명 | 배합 설비 유지보수, 온도·시간 설정 관리 |

**현행 핵심 문제**: 배합비율 결정 권한이 수석 기술원 1인에게 집중되어 있으며, 해당 인원의 부재(출장, 휴가, 퇴직) 시 품질 불안정이 발생하는 구조적 취약점이 존재한다.

## 3. 현행 업무 분석

### 3.1 납땜 합금 배합 공정 현황

#### 3.1.1 현행 프로세스 흐름

납땜 합금 배합 공정은 아래와 같은 순서로 진행된다.

```
[STEP 1] 생산 계획 접수
    - 생산관리팀으로부터 월간/주간 생산 계획 수령
    - 납땜공정팀 팀장이 배합 LOT 수량 및 일정 확인
    - 소요 원재료(SN/AG/CU/PB) 수량 산출 → 구매팀 발주 요청

[STEP 2] 원재료 입고 및 검수
    - 공급사(SUP_A, SUP_B, SUP_C)로부터 원재료 입고
    - 입고 검수: 납품서 수량 확인, 육안 검사
    - 성분 분석 미실시 (현행): 공급사 성적서 신뢰에 의존
    - 창고 보관 (선입선출 미준수 사례 다수)

[STEP 3] 배합비율 결정 (핵심 단계 — 현행 문제 집중)
    - 수석 기술원이 과거 경험과 직관을 바탕으로 배합비율 결정
    - 참고 자료: 수기 작성 노트, 과거 LOT 기록지 (엑셀 파일 일부)
    - 결정 기준: "이전에 잘 나왔던 조건"을 반복 적용
    - 공급사 변경, 원재료 순도 변화 등은 개인 경험으로만 보정
    - 공정 온도(용융 온도): 수석 기술원 판단 (주로 220~270°C)
    - 용융 시간: 수석 기술원 판단 (주로 30~60분)

[STEP 4] 원재료 계량 및 투입
    - 기술원 2명이 수석 기술원의 지시에 따라 SN/AG/CU/PB 계량
    - 계량 도구: 산업용 저울 (분해능 0.1g)
    - 목표 배합비율 대비 실제 투입량 편차: ±0.5~1.5% 발생
    - 계량 오류 기록 없음 (현행): 오류 발생 시 수기 메모 수준

[STEP 5] 용융 및 혼합
    - 배합 용융로에 원재료 투입 후 설정 온도까지 가열
    - 용융 후 지정 시간 유지 (교반 주기는 담당자 재량)
    - 온도 기록: 설비 자체 로그 (별도 추출 미실시)
    - 완료 후 주조 또는 분말 성형

[STEP 6] 성분 분석 및 품질 검사
    - 배합 완료 LOT에서 샘플 채취 (LOT당 3개 시편)
    - XRF 분석기로 SN/AG/CU/PB 성분비 측정
    - 측정값과 목표값 비교 → 품질 검사원 육안 판단으로 합격/불합격 결정
    - 품질 점수 산출: 수식 미정립, 검사원마다 기준 상이
    - 불량 판정 기준: "주요 성분 오차 ±2% 초과" (비공식 기준)

[STEP 7] 결과 기록 및 보관
    - 합격 LOT: LOT 기록지 작성 후 출고
    - 불량 LOT: 수석 기술원에게 보고 → 재배합 또는 폐기 결정
    - 기록 방식: 엑셀 파일에 수동 입력 (표준 양식 없음)
    - 보관: 로컬 PC 저장 (백업 없음, 담당자별 파일 분산)
```

### 3.2 품질 관리 현황

#### 3.2.1 현행 품질 검사 방법

| 검사 항목 | 검사 방법 | 검사 주기 | 합격 기준 |
|-----------|-----------|-----------|-----------|
| 성분비 (SN/AG/CU/PB) | XRF 형광 X선 분석 | LOT당 1회 (3개 시편 평균) | ±2.0% 이내 (비공식) |
| 외관 검사 | 육안 + 확대경 | LOT당 전수 | 이물질, 산화 없음 |
| 용융점 확인 | 시편 용융 테스트 | 월 1회 (랜덤 2 LOT) | 기준값 ±5°C 이내 |
| 기계적 특성 | 인장 강도 테스트 | 분기 1회 | 내부 규격 (비공개) |
| 납품 성적서 확인 | 서류 검토 | 입고 시 | 공급사 자체 기준 |

**현행 품질 검사의 한계**:
- 품질 "점수"를 산출하는 통일된 기준이 없어 검사원마다 판정 기준이 다름
- XRF 측정값은 성분비만 확인하며, 배합 조건(온도·시간)의 영향을 정량적으로 반영하지 못함
- 검사 결과가 다음 LOT 배합에 피드백되지 않음 (단방향 기록)

### 3.3 데이터 관리 현황

#### 3.3.1 수집 데이터 항목

현재 납땜 배합 공정에서 수집되는 데이터 항목 및 수집 방식은 다음과 같다.

| 데이터 항목 | 수집 방법 | 수집 주기 | 저장 위치 | 디지털화 수준 |
|-------------|-----------|-----------|-----------|---------------|

## 4. 현행 시스템 문제점

### 4.1 업무 효율성 문제

**F1. 배합비율 결정 병목 — 1인 의존 구조**

배합비율 결정 전 과정이 수석 기술원 1인의 경험과 판단에 의존한다. 해당 인원의 부재(연평균 출장 15일, 병가 등 포함 약 25일/년) 시 배합 작업이 중단되거나, 경험이 부족한 기술원이 대체 투입되어 불량률이 평균 대비 2.3배 상승하는 패턴이 확인되었다. 지식의 조직 내재화가 전혀 이루어져 있지 않다.

**F2. 배합비율 결정 시간 과다 소요**

최적 배합비율 탐색에 LOT당 평균 45분(최대 2시간)이 소요된다. 과거 경험을 기억하고 조합하는 과정에서 오류가 발생하며, 공급사 변경이나 신규 제품 적용 시 추가 시간이 필요하다. 반면 ML 기반 자동 추천은 수 초 내 결과 도출이 가능하다.

**F3. DOE 실험 계획 수립에 과도한 시간 소요**

새로운 솔더 합금 배합 조건을 탐색하기 위한 실험계획법(DOE) 설계를 수작업으로 수행하며, 설계에 2~5일, 결과 분석에 추가 3~7일이 소요된다. 연간 5~8회 수행되는 DOE의 총 소요 공수는 약 250인시(man-hour)/년으로 추정된다.

### 4.2 품질 관리 문제

**Q1. 사전 예방 불가 — 사후 검사 의존 구조**

현행 시스템은 배합 완료 후 XRF 검사로 불량을 확인하는 사후 대응 구조다. 배합 전 품질 예측이 불가능하여, 불량 LOT가 발생해도 이미 원재료와 공수가 투입된 상태다. ML 기반 사전 예측 시스템 도입 시 배합 전 고위험 조건을 사전에 식별·회피할 수 있다.

**Q2. 공급사별 성분 편차 미반영**

원재료 공급사(SUP_A, SUP_B, SUP_C)별로 원재료 순도와 성분비에 편차가 존재하나, 현행 배합비율 결정 시 이를 체계적으로 반영하지 않는다. 수석 기술원의 경험적 보정에 의존하며, 보정 기준이 문서화되어 있지 않다.

**Q3. 품질 평가 기준 비표준화**

품질 "합격/불합격" 판단 기준이 공식 문서화되어 있지 않으며, 검사원마다 기준이 상이하다. 품질 점수(0~100점)를 정량적으로 산출하는 체계가 없어 공정 개선을 위한 피드백 루프가 작동하지 않는다.

**Q4. 계절·환경 변화 미반영**

배합 공정은 외부 온도, 습도 등 환경 조건의 영향을 받으나, 현행 배합비율 결정 시 이를 반영하지 않는다. 하절기 고온 환경에서의 불량률 증가 패턴이 경험적으로 알려져 있으나 데이터로 검증되지 않았다.

### 4.3 데이터 활용 문제

**D1. 데이터 분산·비표준화로 ML 활용 불가**

LOT 이력 데이터가 담당자별 엑셀 파일에 분산 저장되어 있고, 컬럼명·단위·형식이 표준화되어 있지 않아 ML 학습용 데이터셋 구성에 상당한 정제 작업이 필요하다. 설비 내장 로그(온도, 시간)는 추출조차 되지 않아 주요 피처 데이터가 누락된 상태다.

**D2. 설비 데이터 단절**

배합 용융로에 온도 로그 기록 기능이 있으나 이를 추출하여 품질 데이터와 연계하는 프로세스가 없다. 용융 온도·시간과 품질 간의 상관관계를 데이터로 분석한 사례가 없다.

**D3. 데이터 유실 위험 상시 존재**

로컬 PC에만 저장된 데이터는 PC 고장, 담당자 퇴직, 파일 삭제 등의 사유로 유실될 수 있다. 실제로 2022년 담당자 퇴직 시 약 6개월치 LOT 기록이 유실된 사례가 있었다.

**D4. 데이터 기반 의사결정 문화 부재**

데이터를 분석하여 공정 개선 인사이트를 도출하거나, 이를 의사결정에 활용하는 프로세스가 없다. 품질 이슈 발생 시 데이터 분석보다 경험자 면담에 의존하는 의사결정 방식이 고착화되어 있다.

## 5. 개선 방향 및 기대효과

### 5.1 AI/ML 도입 방향

현행 업무분석 결과를 바탕으로 **Formulation ML 시스템** 도입을 통해 다음과 같은 방향으로 업무를 개선한다.

| 현행 문제 | 개선 방향 | 적용 기술 |
|-----------|-----------|-----------|
| 1인 경험 의존 배합비율 결정 | ML 기반 자동 배합비율 추천 | scikit-learn GradientBoosting + scipy SLSQP 최적화 |
| 사후 불량 검사 | 배합 전 품질 점수 예측 | 회귀 모델 (GradientBoosting, XGBoost) |
| 데이터 분산·비표준화 | 통합 LOT 이력 DB 및 EDA 대시보드 | FastAPI + 구조화 DB, Recharts 시각화 |
| DOE 수작업 설계 | ML 기반 DOE 시뮬레이터 | Full Factorial, CCD, Box-Behnken, Taguchi, LHS |
| 공급사 편차 미반영 | 공급사별 성분 편차 자동 보정 | 피처 엔지니어링 (SUPPLIER_EFFECTS) |
| 설비 데이터 미활용 | 용융 온도·시간 피처 연계 | 설비 로그 추출 + ML 피처 통합 |
| 품질 기준 비표준화 | 정량적 품질 점수 체계 수립 | 0~100점 품질 스코어링 모델 |

**시스템 구성 개요**

```
[공정 조건 입력]          [ML 추론 엔진]              [결과 출력]
온도·시간·공급사  →  GradientBoosting / XGBoost  →  배합비율 추천
성분비 직접 입력  →  SLSQP 최적화               →  품질 점수 예측
LOT 이력 데이터  →  EDA 분석 엔진              →  분포/편차 시각화
실험 계획 파라미터 →  DOE 시뮬레이터             →  최적 조건 탐색
```
