## 1. 문서 개요

본 문서는 Formulation ML 시스템에 대한 사용자 인수 테스트(UAT, User Acceptance Test) 계획 및 실행 결과를 기록한 문서이다.

UAT는 시스템이 실제 업무 환경에서 요구사항을 충족하는지 최종 검증하기 위해 수행되었으며, 제조팀·품질팀·IT팀의 실제 사용자가 직접 시나리오를 실행하였다. 본 문서는 스마트공장 보급·확산사업(선도형 제조혁신 지원사업) 산출물로 활용된다.

---

## 2. UAT 계획

### 2.1 테스트 목적

1. Formulation ML 시스템의 5개 핵심 기능(배합비율 추천, 품질 예측, EDA, 모델 현황, DOE 시뮬레이터)이 요구사항에 부합하여 정상 동작함을 검증한다.
2. 실제 사용자 관점에서 UI/UX의 사용 편의성 및 직관성을 평가한다.
3. 비정상 입력(범위 초과, 합계 오류 등)에 대한 시스템의 오류 처리 적절성을 확인한다.
4. 다양한 환경(데스크탑, 모바일, 다중 탭)에서의 시스템 안정성을 검증한다.
5. 실제 업무에 적용 가능한 수준의 성능 및 응답성을 확인한다.
