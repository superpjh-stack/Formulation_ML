## 1. 문서 개요

### 1.1 목적

본 문서는 **Formulation ML 시스템** 도입 전·후의 기능을 항목별로 비교하여, 스마트공장 보급·확산사업을 통해 실현되는 업무 개선 내용과 기대 효과를 명확히 제시하는 데 목적이 있다.

기능대비표는 다음의 용도로 활용된다.

- 도입 타당성 및 투자 근거 자료
- 검수 기준 문서 (도입 후 기능 구현 여부 확인)
- 사업 성과 보고서 기초 자료
- 이해관계자 커뮤니케이션 자료

### 1.2 비교 기준

| 구분 | 설명 |
|------|------|
| 도입 전 현황 | Formulation ML 시스템 구축 이전의 현행 업무 방식 |
| 도입 후 개선 | Formulation ML 시스템 구축 완료 후 변경되는 업무 방식 |
| 개선 효과 | 정량적 또는 정성적으로 측정 가능한 개선 결과 |

### 1.3 기능 구분 체계

| 구분 코드 | 기능 영역 |
|-----------|-----------|
| BL | 배합 로직 (Blending Logic) |
