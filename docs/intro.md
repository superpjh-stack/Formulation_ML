# 배합비율 최적화 ML — 프로젝트 소개

## 배경

제조 공정에서 원재료의 성분 편차와 공정 조건 변화는 최종 제품 품질에 직접적인 영향을 미친다.
현재 배합비율 결정은 숙련 작업자의 경험과 감(感)에 의존하고 있으며, 이로 인해 다음과 같은
한계가 존재한다.

- 배치(batch)별 품질 편차 발생 → 부적합률 지속 누적
- 원재료 편차에 대한 사전 보정 체계 부재
- 성분·공정·품질 데이터 간 연계 분석 미흡 → 최적 조건 도출 불가
- 공정 중 이상 징후 사전 감지 체계 부재

## 프로젝트 목표

**성분분석 데이터와 공정 데이터를 통합하여, 원재료 편차를 반영한 최적 배합비율을 자동 추천하는 ML 모델을 구축한다.**

| 항목 | 내용 |
|------|------|
| ML 과제 유형 | 회귀(Regression) + 최적화 추천 |
| 핵심 입력 | 성분 분석값(XRF 등), 공정 조건(온도/시간), 원재료 이력 |
| 핵심 출력 | 품질 예측값(불량률/품질등급), 최적 배합비율 추천 |
| 목표 성과 | 부적합률 20~30% 감소 |

## 기대 효과

1. **품질 안정화** — 데이터 기반 배합 결정으로 배치별 편차 최소화
2. **숙련자 의존 탈피** — 작업자 경험이 아닌 모델 추천으로 표준화
3. **선제적 불량 예방** — 원재료 입고 시점에 배합 조건을 사전 계산
4. **Traceability 강화** — 성분→배합→품질 데이터 연결로 전 공정 추적 가능

## 관련 공정 흐름

```
원재료 입고
    ↓
성분 분석 (XRF) ─── [ML 입력: 성분값, LOT 편차]
    ↓
배합비율 결정 ◀─── [ML 출력: 최적 배합비율 추천]
    ↓
용해 공정 ─────── [ML 입력: 온도, 가열시간, 배치정보]
    ↓
주조 / 가공
    ↓
품질 검사 ─────── [ML 피드백: 불량유형, 품질등급]
    ↓
출하
```
